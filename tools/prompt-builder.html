내 업무를
HTML 업무툴로 바꾸는 프롬프트 빌더

반복 업무를 몇 가지 질문에 맞춰 적으면, AI에게 바로 요청할 수 있는 “다운로드 가능한 HTML 파일 제작 프롬프트”를 만들어줍니다.

작성 방법

  1. 업무 이름과 현재 하는 방식을 적습니다.
  2. 툴에서 입력받을 정보와 만들어질 결과를 적습니다.
  3. 프롬프트를 복사해 ChatGPT에 붙여넣습니다.
  4. 생성된 HTML 파일을 다운로드해 브라우저에서 실행합니다.

업무 설명 입력

쉽게 쓰되 구체적으로

회사명 또는 업종. 이 맥락이 결과물에 반영됩니다. (선택)

이 이름이 HTML 파일명으로 사용됩니다.

대표님이 실제로 하는 순서 그대로 적어주세요. “1. 메일 확인 → 2. 엑셀 수정 → 3. PDF 저장”처럼 단계가 보이면 결과물이 덜 흔들립니다.

개발 용어 없이 “어떤 정보를 넣어야 하는지”만 적어도 됩니다.

업체/담당자/날짜 품목/수량/단가 파일 업로드
표/CSV PDF/파일 묶기 메일/체크리스트

디자인, 모바일, 한글, 파일명, 계산 방식 등 원하는 점을 적어주세요.

업무툴 이름과 현재 업무 방식을 먼저 입력해주세요.
입력 내용과 생성된 프롬프트가 발표자와 공유됩니다.

생성된 프롬프트

복사해서 AI에 붙여넣기 
왼쪽 내용을 입력한 뒤 “프롬프트 만들기”를 눌러주세요.
생성된 프롬프트는 ChatGPT, Claude, Gemini 등에 붙여넣어 사용할 수 있습니다. 결과는 코드 설명이 아니라 다운로드 가능한 HTML 파일 하나로 요청하도록 구성되어 있습니다.
